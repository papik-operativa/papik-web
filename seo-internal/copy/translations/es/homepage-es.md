## BLOC 4 · SISTEMA ESKIMOHAUS®

```
[H2]
Eskimohaus®, nuestro sistema constructivo propio

[Párrafo intro]
Después de treinta años construyendo, hemos desarrollado un sistema que combina
los principios del estándar alemán Passivhaus con los materiales y el clima
mediterráneo. No se trata de aplicar una tecnología importada, sino de integrar
la metodología internacional con las particularidades del territorio.

[Subhead]
Cuatro principios sin superfluo

[Grid 4 columnas · sin iconografía, solo tipografía y espacio blanco]
```

### Columna 1 · ENVOLVENTE TÉRMICA

```
Aislamiento continuo, eliminación de puentes térmicos. Reducimos las pérdidas
de energía antes de pensar en generarla, porque la casa eficiente comienza
por el muro.
```

### Columna 2 · HERMETICIDAD CONTROLADA

```
Verificada con test blower-door. Una casa hermética no es una casa que no
respira; es una casa que controla cómo respira.
```

### Columna 3 · VENTILACIÓN CON RECUPERACIÓN

```
Aire renovado constante sin pérdida de energía. Sistemas Zehnder de
recuperación de calor que mantienen la casa saludable durante todo el año.
```

### Columna 4 · ESTRUCTURA EFICIENTE

```
Madera contralaminada (CLT), entramado ligero o sistemas mixtos según
proyecto. Materiales naturales, renovables y con impacto mínimo sobre el
solar.
```

## NOTAS PARA EL DEV

### Toponímia

Sant Cugat del Vallès se mantiene con el nombre oficial catalán también en la versión española (es topónimo oficial). "Vallés Occidental" lleva acento en español. "Cataluña" se usa en lugar de "Catalunya" para coherencia con la versión española y mejor SEO en español.

### Esquema JSON-LD a insertar

Ver `seo-internal/schemas/01-organization-homepage.json`. Reemplazar todos los `{{ ... }}` con valores reales antes de publicar. La versión española mantiene los mismos schemas que la catalana, ya que el `inLanguage` se ajusta automáticamente.

### Resto de notas técnicas

Las notas de imágenes, animaciones, accesibilidad, performance y checklist pre-publicación son idénticas a la versión CA. Ver `homepage-ca-v2.md` para el detalle.
